# 🚀 CTF Data Tables

## 🛡️ Cybersecurity Reference Tool

CTF Data Tables is a cybersecurity web reference tool that helps users quickly identify encoding, hashing, and file signatures used in CTF challenges and cybersecurity analysis.

---

## 📚 Tables Included

### 🔹 Encoding
- Base2  
- Base8  
- Base16 (Hex)  
- Base32  
- Base64  
- Base64URL  

### 🔹 Hashing
- MD5  
- SHA1  
- SHA256  
- SHA512  
- bcrypt  

### 🔹 Forensics / File Signatures
- Image formats  
- Audio formats  
- Video formats  
- Archive formats
